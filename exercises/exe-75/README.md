# Using the Terraform Console

- Look at the **play-data.tf** file in this lab.
- Run the console:  
**terraform console**  

Try the Following expressions:

- **3 + 5**
- 4 * 9
- -6
- 1 + 2 * 3
- 15 / 2
- 23 % 5
- (3 + 4) == 7
- (3 + 4) == 8
- 5 > 3
- ! (5 > 3)
- (5 > 3) && (6 < 3)
- (5 > 3) || (6 < 3)
